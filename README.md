# ExperimentBot
## Normal Commands

### .dice [x]
Rolls an [x]-sided die.
### .rate [x]
Rates [x] from 1-5.
### .hello
Says hello to you!
### .randomuser
Says a random username.
### .help
Links back here.
## Coin Commands

### .coin
Says how many coins you have. You start with 100.
### .coin flip
Flips a coin, you either win or lose 10 coins.
